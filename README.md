# 🎮 Jogo da Velha em C

Este projeto foi desenvolvido como parte dos meus estudos de Programação em C durante o 2º semestre do curso de Sistemas de Informação.

O objetivo foi colocar em prática os conteúdos aprendidos até o momento, principalmente funções, ponteiros, matrizes, estruturas de repetição e estruturas condicionais. Durante o desenvolvimento procurei dividir o programa em funções para deixar o código mais organizado e facilitar futuras manutenções.

Foi um projeto desafiador, principalmente na implementação da lógica do jogo e na verificação das condições de vitória, mas também foi um dos exercícios que mais contribuíram para o meu aprendizado na linguagem C.

## ✨ Funcionalidades

- Menu principal
- Jogo contra o computador
- Escolha das jogadas pelo jogador
- Jogadas aleatórias do computador
- Verificação de vitória
- Verificação de empate
- Estatísticas das partidas
- Exibição do tabuleiro durante o jogo
- Validação das entradas do usuário

---

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Biblioteca `stdio.h`
- Biblioteca `stdlib.h`
- Biblioteca `time.h`

## 📚 Conceitos praticados

Durante este projeto utilizei diversos conceitos estudados na faculdade, como:

- Funções
- Ponteiros (parâmetros por referência)
- Matrizes
- Estruturas condicionais
- Estruturas de repetição

## ▶️ Como executar

Compile o programa:

```bash
gcc jogo_da_velha.c -o jogo
```

Execute:

```bash
./jogo
```

## 💡 O que aprendi

Esse projeto me ajudou a entender melhor como dividir um programa em várias funções e como utilizar matrizes para representar um tabuleiro.
